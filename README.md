# 🎵 MEWA Music

**Музыка без цензуры, рекламы и подписок.**

MEWA — это независимый музыкальный сервис, созданный для тех, кто устал платить за ВК Музыку, Яндекс Музыку или Spotify. Здесь можно слушать любую музыку бесплатно, без рекламы и ограничений.

---

## ✨ Основные функции

- **Полноэкранные субтитры** — текст песни открывается на весь экран, как в караоке.
- **Вход через MEWA или Google** — быстрая регистрация без лишних шагов.
- **Личный профиль** — статистика прослушиваний, любимые артисты и треки.
- **Страницы артистов** — с анимацией биографии и удобным плеером.
- **Заявка для музыкантов** — любой исполнитель может добавить свои треки бесплатно, без платных дистрибьюторов.
- **Оптимизация** — быстрая работа на компьютерах и телефонах.

---

## 🛠️ Стек технологий

- **TypeScript / JavaScript** — основная логика
- **React** — интерфейс и компоненты
- **Tailwind CSS** — стили и дизайн
- **Vite** — сборка проекта
